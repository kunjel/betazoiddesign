10 Foot UI

Back to projects

Client

NDA

Details

  • Situation Design an interface for television to enable effective and quick channel surfing. Search shows more effectively. The interface had to be simple, clean and viewable from a distance of 10 feet.

  • Strategy
    • Design an interface to be used with a remote control device. Make it a tv-like experience page layout. Must be easily viewable at the common tv watching distance. Take into consideration technical aspects like overscan and padding.Keep the color scheme neutral and prevent white text halos.
  • Result
    • Clean, simple interface what is easy to navigate without much learning curve involved. Clean, strong visual cues.
